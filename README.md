# Info
Утилита для запуска уже существующих ВМ в Yandex Cloud.
При запуске читает список ВМ из переменных окружения и последовательно запускает их.

# How-To
1) Создаёте сервисную учётную запись с ролью в каталоге `compute.operator`
2) Добавляете учётной записи авторизованный ключ и сохраняете открытый и закрытые ключи локально в файл, будет JSON.
3) С помощью base64 кодируете файл (cat <my-file.json> | base64 > my-file.enc)
4) Помещаете содержимое файла my-file.enc в переменную окружения `YC_KEY_DATA` 
5) Создаёте ВМ в YC, id новой ВМ сохраняете в переменной окружения `YC_VMS`. Для нескольких ВМ использовать `,` без пробелов
6) Запускаете приложение, если всё хорошо - будет подобная запись
   ```
   time="2026-04-03T11:44:27Z" level=info msg="bg0msi9f35ukv8: RUNNING"
   ```

# Settings
- `YC_KEY_FILE` - Абсолютный путь к JSON файлу с данными для авторизации под сервисной учётной записью
- `YC_KEY_DATA` - Закодированный base64 JSON
- `YC_LOG_LEVEL` - Уровень логирования (info, debug)
- `YC_LOG_FILE` - Файл для логов, по-умолчанию - /var/log/vmStarter.log
- `YC_VMS` - Список ВМ для запуска, разделитель `,`
